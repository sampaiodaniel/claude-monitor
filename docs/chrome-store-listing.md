# Chrome Web Store Listing

## Name
Claude Monitor

## Short Description (132 chars max)
Monitor your Claude AI usage in real time. Get alerts before hitting limits. Track usage history with charts.

## Detailed Description

Stop refreshing the Claude usage page! Claude Monitor puts your usage right on the Chrome toolbar.

REAL-TIME MONITORING
- Color-coded badge showing current session usage percentage
- Popup with session and weekly progress bars
- Countdown timers showing when limits reset
- Updates automatically every 5 minutes (configurable)

SMART ALERTS
- Windows notifications at customizable thresholds (default: 50%, 70%, 90%)
- Alerts fire once per threshold per reset cycle - no notification spam
- Works for both session (5-hour) and weekly limits

USAGE HISTORY
- Automatic logging at the end of each session window
- Interactive bar chart showing usage per session, grouped by day
- Paginated table with day grouping and color-coded values
- Summary stats: average session, total sessions, weekly average

FULLY CONFIGURABLE
- Custom color intervals for usage ranges (8 color palette)
- Custom alert thresholds
- Adjustable polling interval (1-30 min)
- Optional Sonnet usage display

ZERO SETUP
- No API key needed - uses your existing Claude session
- Auto-detects your organization
- Just install and go

PRIVACY FIRST
- All data stays in your browser (chrome.storage)
- No external servers, no analytics, no tracking
- Open source: github.com/sampaiodaniel/claude-monitor

Works with Claude Pro, Team, and Enterprise plans.
